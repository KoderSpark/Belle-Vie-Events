# Brand Guidelines & Social Media Resources

This directory contains comprehensive guidelines for maintaining brand consistency across all visual and content materials for **haus of veloura Events**.

## 📋 Contents

### Brand Guidelines (`BRAND_GUIDELINES.md`)
Complete brand identity guidelines including:
- **Color Palette:** Warm Taupe Brown (#89725B) and Black (#000000)
- **Color Ratios:** 60% White, 25% Black, 15% Taupe
- **Design Principles:** Minimalist aesthetic, clarity, elegance
- **Typography:** Font pairings and hierarchy
- **Visual Elements:** Lines, borders, buttons, image styles
- **Do's and Don'ts:** Quick reference for brand compliance

### Social Media Guidelines (`SOCIAL_MEDIA_GUIDELINES.md`)
Platform-specific strategies and templates for:
- **Instagram:** Post templates, content pillars, engagement strategy
- **Facebook:** Event promotions, community content
- **Stories & Reels:** Visual templates and format guidelines
- **Caption Writing:** Best practices and formulas
- **Hashtag Strategy:** Branded, campaign, and industry hashtags
- **Performance Metrics:** KPIs and tracking guidelines
- **Content Calendar:** Template for scheduling

### Brand Colors (`src/lib/brandColors.ts`)
Developer-ready color constants for:
- TypeScript/React usage
- Tailwind CSS configuration
- CSS variables
- Color ratio calculations

## 🎨 Quick Color Reference - Dark Theme

**Three Brand Colors:**
- **Dark Background:** `#000000` (0, 0, 0) - Primary background
- **Warm Taupe Brown:** `#89725B` (137, 114, 91) - Accent highlights
- **White:** `#FFFFFF` (255, 255, 255) - Text & light accents

**NO OTHER COLORS SHOULD BE USED**

## 📱 Social Media Post Templates

### Instagram Post Formula
```
Dark background (#000000)
Taupe (#89725B) accent elements
White text overlay
```

### Facebook Ad Formula
```
Dark background or image with dark overlay
White text/copy
Taupe (#89725B) accent elements
```

## ✨ Key Brand Principles - Dark Theme

1. **Minimalist Aesthetic** - Clean lines, maximum dark space
2. **Elegance Through Restraint** - Quality over quantity
3. **Clear Communication** - Simple, readable, professional with white text
4. **Consistent Application** - Uniform dark theme across all touchpoints
5. **Strategic Color Harmony** - Dark background, taupe accents, white text

## 🚀 Getting Started

### For Content Creators:
1. Read `BRAND_GUIDELINES.md` for visual principles
2. Check `SOCIAL_MEDIA_GUIDELINES.md` for platform-specific guidance
3. Use the templates provided for each post type
4. Maintain the 60/25/15 color ratio in graphics

### For Developers:
1. Import colors from `src/lib/brandColors.ts`
2. Add brand colors to Tailwind CSS config (already included)
3. Use consistent color variables across components
4. Refer to brand guidelines for UI element styling

## 📊 Social Media Best Practices

- **Posting Frequency:** 3-4 Instagram posts per week, 2-3 Facebook posts
- **Best Times:** Weekdays 10 AM & 3 PM IST (Instagram), 9 AM, 1 PM, 6 PM (Facebook)
- **Engagement:** Respond to comments within 6 hours
- **Hashtags:** 10-15 per Instagram post (appears cleaner than 20+)
- **Caption Length:** Keep first line under 150 characters for impact
